# OceanMooc

#### 项目介绍
该项目为作者毕业设计项目群的的Android客户端。
项目刚刚启动，计划于6月底完成基础库和、及部分业务逻辑编写。

#### 软件架构
软件架构说明
ocean-annotations 元注解库
ocean-compiler    编译时注解处理
ocean-core        公共核心库
ocean-mooc        mooc业务module
